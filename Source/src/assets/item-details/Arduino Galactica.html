My Game (straight from the project proposal)

The player will control the ship by moving the joystick left and right. The player is locked to exclusively horizontal movement. The player can click the button to shoot a laser and can press the joystick down to shoot a big laser, which takes time to charge up. The game will be broken up into levels and scores will be given based on levels completed and speed at which the aliens were defeated, with less time being worth more points. Rather than the classic rows of aliens zig zagging down from the top, there will be intermittent aliens generated randomly (with a scaling difficulty) and there will also be different power ups that the aliens sometimes drop. Instead of the aliens moving closer to the bottom, they will move erratically, periodically shooting at the player. The levels will follow the naming scheme of “1-1, 1-2, …, 5-3” and so on. Each “world” will have 3 levels and will have a different visual theme. When the player shoots, gets shot, defeats an enemy, etc, there will be a small vibration to create an immersive feel. The main menu of the game will prompt the player to press the start button to begin, and display the highest level that was completed. The hardware components I used for this project are three LEDs, a 4 digit 7 segment display, a buzzer, an LCD screen, which I communicated with using SPI, a shift register, two buttons, and a joystick. Since I have experience with game development, I tried to make the base of my game as similar to a real game engine as possible. Every game tick, it would step all the entities, including the player, enemies, and projectiles. I had also built some sketchy functions for outputting sprites to the LCD, which is basically just predefining bitmap size functions. I’m sure there is a much more elegant way to do that, but that would require me learning in depth C++ and I didn’t have time for that. The code for this game ended up being about 900 lines and it is all posted on my github. I also uploaded the video demonstration to youtube and it got thousands of views.
